✍️ HOW TO WRITE A CAMPAIGN BRIEF
Brief BRIEF tab mein paste karo. AI Parse ya Quick Parse (Offline) use karo. Neeche format hai.
IDEAL FORMAT
[Brand Name] brief — need [Number] [Niche] [Gender] creators from [City] for [Platform], [Follower range] followers
Budget: ₹[Amount] per reel | Deliverables: [What they need to post]
EXAMPLES
ENGLISH
Raymond brief — need 10 male fashion creators from Mumbai for Instagram, 50K-500K followers. Budget ₹20,000 per reel.
HINGLISH
Nike ka campaign hai, 15 fitness creators chahiye Mumbai aur Delhi se, Instagram ke liye, 100K se upar followers.
SIMPLE / MINIMAL
Nykaa, beauty, female, Mumbai, Instagram, 50K-300K
✅ Always mention: Brand Name, Niche, Platform, Location, Creator Count
✅ Follower range format: 50K-500K or 50K to 500K or 1L-5L
✅ Gender: male / female / ladka / ladki sab work karta hai
✅ Cities: Mumbai, Delhi, Bangalore, Pune, Hyderabad etc.
⚠️ Platform name (Instagram) ko brand ki jagah mat likho

💡 BEST PRACTICES — Maximum Results
DATABASE TIPS
📦 ZIP upload: 50 files ek ZIP mein daal ke upload karo — ek hi click mein sab process
🔁 Duplicate handling: Same creator naam se duplicate automatically remove hoga
📋 Column name: Sirf "Name" column required hai — baaki sab optional
📊 Sheet tabs: Summary, Dashboard, README named sheets automatically skip hoti hain
💰 Cost date: "Cost Per Reel" ke saath "Cost Date" column bhi rakho — e.g. "Jan 2025" — taaki pata rahe kab ka rate tha
👤 POC tracking: POC Name + POC Agency + Last Contacted + Status — in 4 columns se pata rahega last mein kisse baat hui thi
SHORTLIST TIPS
🎯 Start broad: Pehle sirf niche filter karo — phir location + followers add karo
📍 Location tip: City name exactly likho (Mumbai not Bombay) — ya blank chhod do
📊 Score ≥70%: Green score wale creators best match hain — prefer karo
✅ Select creators: Checkbox pe tick karo → "Export Selected" se sirf selected ki XLSX download
🔢 Count limit: "Need X creators" field mein number dalo — default 100 hai
BRIEF TIPS
🤖 AI Parse: Settings mein Groq free API key dalo → AI automatically sab extract karega
⚡ Quick Parse: Internet nahi hai? "Quick Parse (Offline)" use karo — regex se parse hoga
📱 WhatsApp message: Guide tab → WhatsApp→AI slide → paste karo → ready prompt milega
🌐 Any language: Hindi, Hinglish, Marathi, English — sab kaam karega
EXPORT TIPS
📥 XLSX export: Yellow header, styled — directly client ko bhej sakte ho
📤 Import Log export: Database tab mein "Export Log" button — CSV download hoti hai with all errors
