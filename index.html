💡 AI PROMPTS LIBRARY
Ready-made prompts — copy karke ChatGPT, Claude, ya kisi bhi AI mein paste karo. Apna kaam ho jaayega.
📋 BRIEF PARSER PROMPT
WhatsApp/email brief ko structured format mein convert karo
You are a campaign brief parser for influencer marketing. Extract the following fields from the brief text I give you and return ONLY a JSON object (no markdown, no explanation): { "brand": "", "niche": [], "platform": "", "location": "", "min_followers": 0, "max_followers": 0, "gender": "", "budget_per_reel": "", "deliverables": "", "campaign_start": "", "special_requirements": "" } Rules: - niche: array of niches like ["fashion", "fitness"] - platform: "Instagram" | "YouTube" | "Both" | "" - gender: "Male" | "Female" | "Any" | "" - followers: integer (50K = 50000, 1M = 1000000) - If not mentioned, use empty string or 0 Brief: [PASTE YOUR BRIEF HERE]
📨 OUTREACH MESSAGE PROMPT
Creator ko pehla WhatsApp message — friendly, professional
Write a short, friendly WhatsApp outreach message to an influencer for a brand collaboration. Keep it conversational, not too formal, under 80 words. Include: - Brand name and campaign type - Why I'm reaching out to them specifically - What we need (deliverable) - Soft CTA to check interest Details: - Brand: [BRAND NAME] - Creator name: [CREATOR NAME] - Niche: [NICHE] - Deliverable: [e.g. 1 Reel] - Budget: [BUDGET] - My name/agency: [YOUR NAME / AGENCY] Language: Hinglish (mix of Hindi and English) Tone: Friendly, direct, not salesy
✉️ CONFIRMATION MAIL PROMPT
Deal confirm hone ke baad formal email
Write a professional confirmation email to an influencer after a deal is finalized. Include all campaign details clearly. Keep it formal but warm. Campaign Details: - Brand: [BRAND NAME] - Creator: [CREATOR NAME] - Deliverable: [e.g. 1 Instagram Reel + 1 Story] - Go-live Date: [DATE] - Budget: ₹[AMOUNT] - Payment Terms: [e.g. 50% advance, 50% post-delivery] - Script/Brief: [ATTACH OR SUMMARIZE] - Usage Rights: [e.g. Brand can repost on their handle] - My name: [YOUR NAME] - Agency: [AGENCY NAME] Format: Subject line + email body. Professional English. End with signature block.
📊 SHEET FORMAT PROMPT
Unstructured creator data ko proper Excel format mein convert karo
Convert the following raw influencer data into a clean Excel-ready CSV format. Use these exact column headers: Name, Niche, Instagram Link, IG Followers, YouTube Link, YT Subscribers, Gender, Location, Contact, Email, Cost, POC Name, POC Agency, Notes Rules: - One creator per row - Followers: plain numbers only (e.g. 245000 not "245K") - Links: full URLs (https://instagram.com/handle) - If data missing: leave cell empty - No currency symbols in Cost column Raw data to convert: [PASTE YOUR RAW DATA HERE]
🔍 CREATOR RESEARCH PROMPT
Ek creator ke baare mein quick analysis karo
Analyze this influencer profile and give me a quick brand-fit report: Creator: [NAME / HANDLE] Platform: [Instagram/YouTube] Niche: [NICHE] Brand I want to promote: [BRAND] Brand category: [e.g. FMCG, Finance, Fashion] Tell me: 1. Is this creator a good fit for the brand? (Yes/No + reason) 2. Estimated engagement quality (High/Medium/Low) based on niche 3. Any red flags to check (fake followers, controversial content, etc.) 4. 2-3 things to verify before finalizing 5. Suggested deliverable format for this niche Keep it brief — bullet points preferred.
💰 NEGOTIATION MESSAGE PROMPT
Creator ka rate zyada ho toh politely negotiate karo
Write a polite WhatsApp message to negotiate an influencer's rate. Be respectful, don't lowball rudely. Offer a counter and keep the door open. Situation: - Creator asked: ₹[THEIR RATE] - My budget: ₹[YOUR BUDGET] - Deliverable: [e.g. 1 Reel] - Brand: [BRAND NAME] - Any extra we can offer: [e.g. product gifting, exclusivity, long-term collaboration] Tone: Respectful, professional, Hinglish Length: Under 60 words
